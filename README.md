# SmartLibraryPlus

Hibernate ORM tabanlı Akıllı Kütüphane Sistemi.

## Kullanılan Teknolojiler
- Java
- Hibernate ORM
- SQLite
- Maven

## Proje Açıklaması
Bu proje bir üniversite kütüphanesinde kitapların ve öğrencilerin yönetilmesi, kitapların ödünç verilmesi ve geri alınmasını sağlayan konsol tabanlı bir sistemdir.

Sistem Hibernate ORM kullanarak SQLite veritabanı üzerinde çalışır.

## Çalıştırma
1. Projeyi GitHub’dan indirin
2. IntelliJ IDEA ile açın
3. Maven bağımlılıklarının inmesini bekleyin
4. `Main.java` dosyasını çalıştırın

SQLite veritabanı otomatik olarak oluşturulur.

## Özellikler
- Kitap ekleme ve listeleme
- Öğrenci ekleme ve listeleme
- Kitap ödünç verme
- Kitap iade etme
- Ödünç listesini görüntüleme
